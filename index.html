Thoughts on Progressive Enhancement

Progressive enhancement can be seen as starting with the necessities, which in the case of web development is the content. It does not matter how pretty a website looks if the content is not available to users. The PE method helps make sure the content is on the page for all users before decorating it with CSS. It is important that modern web developers use this method since the number of screens that a website can be displayed on is at its greatest right now. The processing power from screen to screen can differ so it is best that at least the content is getting to the user rather then the glam.
